# SQL_Bootcamp

## Описание проекта

**SQL_Bootcamp** — учебный проект, посвящённый основам работы с **реляционными базами данных** и языком **SQL**.  
Проект помогает шаг за шагом разобраться, как хранить, извлекать и анализировать данные, а также как проектировать структуру базы и оптимизировать запросы.  

Главная цель — научиться уверенно работать с базами данных: от создания таблиц до написания сложных запросов с объединениями, подзапросами и агрегатными функциями.  
В результате выполнения заданий формируется целостное понимание того, как данные организуются, обрабатываются и связываются в реальных проектах.

### Цели проекта

- Освоить базовые принципы реляционной модели данных.  
- Научиться формулировать корректные и эффективные SQL-запросы.  
- Разобраться с видами связей между таблицами и нормализацией данных.  
- Получить представление о транзакциях, индексах и оптимизации запросов.  
- Практически закрепить материал на примере работы с демонстрационной базой данных.

### Основные темы

- Создание и изменение таблиц (DDL)  
- Добавление, обновление и удаление данных (DML)  
- Выборка данных с фильтрацией, сортировкой и агрегацией  
- Соединения таблиц (JOIN) и подзапросы  
- Ограничения целостности, ключи и индексы  
- Базовые принципы оптимизации запросов  

### Итог

Проект даёт прочную основу для понимания принципов работы баз данных и уверенного владения SQL.  
Участник учится видеть структуры данных «изнутри», анализировать взаимосвязи и писать понятные, эффективные запросы — важный навык для любого разработчика и аналитика.
